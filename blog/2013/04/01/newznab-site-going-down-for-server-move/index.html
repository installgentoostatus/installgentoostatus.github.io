Newznab Site Going Down for Server Move

Apr 1st, 2013

Today the newznab usenet indexer at news.installgentoo.com will be going down for a server move.

I have gotten it it’s own server to help alleviate the large amount of resources it has been taking up on the server it shared with the tracker.

With this move I will be starting mostly fresh, and not migrating over any of the previous releases.  I will be moving over the user database so users of the previous install will not have to be reinvite and API keys will stay the same.

 

I will post back once the migration is complete.
