FTP Move

Jun 4th, 2013

We have successfully moved our FTP service over to a new server.  The new server has the same specs but now will be running FreeBSD 9.1 instead of Debian GNU/Linux.  The DNS change may take a bit of time to update, so bear with.  Once DNS has propagated I will make a new thread announcing the changes.

Once I get some time I will make the old FTP server available for shells, since the server is paid for until the end of June.  It will not be continued past the expiration date though.
