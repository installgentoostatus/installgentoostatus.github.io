Changes to Our Image Services

Apr 29th, 2013

Ok I have been adding and removing and changing image services for the past few weeks.  Since I instated it we have been using img.installgentoo.com as a basic image dump and galleries.installgentoo.com as a galleries site.

The image dump site was cheveretto based and not free as in freedom, it also while very nice lacked a galleries part.

The galleries site did not show much use and was coppermine based.

I am going to consolidate these back into img.installgentoo.com which is now CF Image Host based, an open source image hosting solution that does have support for private images and a public, taggable, searchable gallery.


 

I plan on trying to bulk import the previous images but they will be at a different url so stay tuned.
