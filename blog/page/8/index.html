Playing With Windows Azure

Apr 21st, 2013

So for a few weeks I have been setting up a few “fail-safe” sites to monitor the servers and to keep users up to date in the event of an emergency.

So I experimented with a few shitty (free) web hosting providers as well as setting up a blog.  Now that we have gotten used to them, I have moved them.

To a free tier account on Windows Azure, cloud services.

TL;DR

URLs are changing for the status stuff, its going to be faster and more reliable now.

So now the blog is at:

http://installgentoostatus.azurewebsites.net/

and server monitoring is at:

http://installgentoostatus.azurewebsites.net/status

 

Image Gallery Changing

Apr 21st, 2013

After experimenting with piwigo for about a week I have closed media.installgentoo up.  I did not like piwigo for a multi-user setup as it lacked some basic access control settings.  Aside from that (in a single user setup) it works very well and did have some nice plugins.

NulledNets and myself have setup galleries.installgentoo.com now.  It is up and ready to go, and is based off the opensource gallery suite Coppermine.

So please feel free to create albums!
