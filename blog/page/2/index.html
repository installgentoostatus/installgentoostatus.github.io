Big Update

Sep 13th, 2013

I have been very, very busy as of late but have managed to make several changes behind the scenes.  This includes adding services, changing services, and removing services.  I thought it would be best to explain in a post all of the changes instead of several.

Changes:
Last week we successfully moved the ftp, tracker, and blogs to a much more powerful server.  So you should notice those services are quite quicker.

Things going away/already gone:

  • http://chan.installgentoo.com - not enough interest
  • http://radio.installgentoo.com - icecast is really great but not enough interest to dj and listen



New things:

  • http://status.installgentoo.com - is now our blogger status blog, status was running on an amazon ec2 instance but now that our free tier is running out we needed to find an alternative.  
  • http://servers.installgentoo.com - the 2nd half of our status site was the server checker, this has been moved off of ec2 and installed on a free web host (for now).
  • mumble.installgentoo.com - now is set for max audio quality 
  • http://music.installgentoo.com - is back by popular demand, still in beta testing but this is now a subsonic instance streaming the ftp music folder.
  • http://archive.installgentoo.com - is back as well but this time as a nice foolfuuka archiver, for now we are just archiving /g/ those this will probably change.  Also on this site is /IG/ and a spot for our own chan (like chan.IG), I will probably redirect chan.installgentoo.com here.
  • http://bb.installgentoo.com - a more persistent form of communication and community is being tested and tweaked, so this is coming soon.
  • New ftp status pages: http://ftp.installgentoo.com/vnstat/ and http://ftp.installgentoo.com/sysinfo/
  • http://etherpad.installgentoo.com - just another etherpad instance
  • nntp://news.installgentoo.com - our own private usenet server (local groups only), I will be making a tutorial on how to connect soon.


Well, I think that about covers everything.
