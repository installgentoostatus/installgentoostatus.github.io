Ftp Issues

May 31st, 2013

it has been brought to my attention that the ftp server was unreachable via http and ftp for awhile (just today?).  The issues seem to stem from insane http daemon memory usage and insane amounts of connections via ftp from one IP.

These issues have been corrected and the services are operating normally.  To prevent this from happening again I have limited the number of concurrent ftp connections from one IP from unlimited down to 6.
